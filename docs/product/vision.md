# SmartBursar Vision

SmartBursar is a reliable desktop finance system for Ugandan schools. It exists to help bursars, headteachers, and school proprietors manage school money without relying on unstable spreadsheets or manual cashbooks.

## Core outcomes

- enable accurate student fee management and payment collection
- make cashbook entries automatic and auditable
- provide clear reports for daily collections, student balances, and term summaries
- support printing receipts and reports in A4 format
- run offline on low-spec Windows computers

## What success looks like

- the bursar can record payments and expenses in minutes
- the system produces usable financial reports without manual reconciliation
- the app feels simple and predictable to use
- maintenance is easy because the code is easy to understand

## Guiding philosophy

- Product first: every feature must solve a real school need.
- Efficiently Functional Simplicity: keep code and workflows as simple as possible.
- Learn while building: document architecture and choices as part of development, not as separate coursework.
